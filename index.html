The Three main diffrences between web1,web2 and web3
1. Web 1.0 consisted of mostly personal pages that were completly static, with web 2.0 websites became more dynamic and things like visit counters and dynamic presentation of data to each user. with web 3.0 this trend grew even more and things like reactive interfaces and live data became more prominent.
2. Web 1.0 you would only be able to search a specific url while in web 2.0 you can search of a webpage using search engines that index the contents of the web's tags. web 3.0 enables better search results using more complex indexing and searching algorithms that can infer semantic connections and meanings.
3. Web 1.0 the content can be changed only by its author. Web 2.0 is read/write, and Web 3.0 is also analyzing the data to provide knowledge
